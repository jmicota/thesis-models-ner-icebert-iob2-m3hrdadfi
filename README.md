# NER model checkpoints for anonymization of Icelandic patient-to-clinic messages

This is a set of checkpoints for Icelandic NER RoBERTa model (https://huggingface.co/m3hrdadfi/icelandic-ner-roberta) sequentially fine tuned on synthetic Icelandic messages from patients to clinics. The data used for training the model can be found here: https://github.com/jmicota/thesis-datasets-ner-icebert-iob2
